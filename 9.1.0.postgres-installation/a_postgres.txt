# postgres for windows 

1. Download the Installer 
    a. Visit the official PostgreSQL Windows Download Page.
    b. Click the link for "Download the installer"; this will redirect you to the EDB website where the actual files are hosted.
    c. Select the latest version (e.g., PostgreSQL 18) for the Windows x86-64 architecture.

2. Run the Installation Wizard 
    a. Locate the downloaded .exe file and double-click to run it as an Administrator.
    b. Installation Directory: You can keep the default path (usually C:\Program Files\PostgreSQL\<version>) or specify a new one.
    c. Select Components: It is recommended to keep all defaults selected:
            PostgreSQL Server: The core database engine.
            pgAdmin 4: The graphical management tool.
            Command Line Tools: Essential for using psql.
    d. Data Directory: This is where your actual data will be stored. The default is usually fine.
    e. Set Password: Create a password for the database superuser (the default user is postgres). Do not forget this password, as you will need it to log in.
    f. Port: The default port is 5432. Change this only if you already have another service using it.
    g. Advanced Options: Keep the default locale.

3. Verify the Installation
    Open pgAdmin 4: Search for "pgAdmin" in your Start menu. When it opens, you may be asked to set a "Master Password" for the application itself.
